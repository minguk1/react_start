# REACT
## 기초
1. 실습 환경 및 시작
    리액트 설치
    Node.js 설치
    VS Code 사용

    VS Code 실행 후 npx create-react-app . 명령어 입력
    전체적인 코드 작성후 npm start 명령어 입력

    index.js 파일과 App.js, App.css 파일을 중심으로 작성
